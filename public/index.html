① 기간 고르기 (맨 위 버튼)
오늘·어제는 그 하루, 최근7일·이번달·지난달은 그 기간, 기간 설정은 직접 날짜 범위를 정해요. 고르면 모든 탭이 그 기간으로 바뀝니다. (전체·네이버 탭은 '하루' 기준이라, 여러 날을 고르면 마지막 날만 표시돼요.)
📅 과거 데이터, 언제까지 볼 수 있나요?
광고 매체마다 자료를 보관하는 기간이 달라요. 그 기간보다 더 옛날 날짜를 고르면 숫자가 비어 보여요(고장 아님).
네이버 검색광고최대 약 1년(365일)
META (인스타·페북)최대 약 3년(37개월) · 단 상세 분석은 12~13개월
카카오모먼트최근 약 2년 (한 번에 최대 31일씩 조회 · 당일 제외)
크리테오최대 약 2년
유입·매출 (Cafe24)쇼핑몰 분석 제공 기간 기준
📥 GFAAPI 없음 → 엑셀 올린 만큼 (제한 없음)
👉 가장 짧은 네이버(1년) 기준으로, 최근 1년 안이면 모든 매체가 함께 보여요. 그보다 옛날은 매체마다 차이가 있어요.
② 탭별로 보는 것
  • 📊 전체 — 모든 매체를 한 표로(광고비·전환매출·ROAS) + 광고 잔액 + 일자별 추이 그래프
  • 🔗 유입·매출 — 쇼핑몰(yogibo.kr)의 실제 주문·매출을 광고 매체별로 (정산 기준 진짜 매출)
  • 네이버 — 검색광고 캠페인별 성과. 행을 클릭하면 광고그룹 → 키워드까지 펼쳐져요
  • META — 인스타·페북 캠페인별. 행 클릭 → 소재별, 썸네일 클릭 → 실제 광고 이미지 크게
  • 카카오 — 카카오모먼트 캠페인·메시지(플친) 성과 + 비즈월렛 잔액
  • 크리테오 — 광고주 승인 완료 시 자동 표시 (현재 승인 대기)
  • 📥 GFA — 엑셀(.xlsx)을 올리면 날짜·캠페인별로 정리해서 보여줘요
  • 🤖 AI 분석 — 기간을 정하면 AI가 그 기간 광고 성과를 평가하고 개선 방향을 제시
③ 자주 나오는 용어 (쉽게)
  • 광고비 쓴 돈 · 노출 광고가 보인 횟수 · 클릭 눌린 횟수
  • CTR 클릭÷노출(관심도) · CPC 클릭 1번에 든 비용
  • 구매 / 구매매출 실제 결제 건수 / 결제 금액
  • 구매 ROAS 광고비 대비 매출 — 예: 500%면 광고비 1원이 매출 5원. 높을수록 좋고 300% 이상이면 우수
  • 🛒 장바구니 담기(아직 안 산 것) — 실매출 아님, 참고용으로 따로 표시
④ 팁
표의 캠페인·광고그룹 행을 클릭하면 더 자세히 펼쳐지고, META 소재 썸네일을 클릭하면 실제 이미지가 크게 떠요. 숫자가 비어 보이면 위에서 기간을 바꿔보세요.
⑤ 🤖 AI 분석 — 분석자료로 활용하기
AI가 선택한 기간의 모든 광고 데이터(네이버·META·Cafe24 실매출·파워링크 키워드·비즈머니 잔액)를 모아 성과 평가와 개선 방향을 글로 정리해줘요. 그대로 주간보고·회의 자료로 쓸 수 있습니다.
쓰는 순서
  1. 맨 위에서 기간 선택 (예: 지난주·이번달)
  2. AI 분석 탭의 "무엇을 분석할까요?" 칸에 보고 싶은 내용 입력 (비우면 전체 종합)
  3. 🔍 분석하기 클릭 → 잠시 후 결과
이렇게 물어보면 좋아요 (예시)
  • "메타 소재별로 뭐가 잘됐고 뭘 끄는 게 좋을까"
  • "광고비를 줄여도 되는 곳을 찾아줘"
  • "파워링크 키워드 위주로 ROAS 점검해줘"
  • "카카오 플친 성과는 어때?"
  • "지난주 대비 개선점과 다음 주에 할 액션"
결과 구성
총평 → 잘된 곳 → 점검할 곳 → 🧭 방향성 → 바로 할 액션 3가지
비용·캐시 (걱정 마세요)
  • 1회 약 ₩수십~수백 — 화면에 토큰·예상비용이 표시돼요
  • 같은 기간 + 같은 질문은 다시 봐도 무료(저장됨). 새로 만들려면 🔄 다시
  • 팁: 기간이 길수록 비용↑ · 질문을 구체적으로 쓸수록 더 쓸모있는 답이 나와요
💡 광고 매체가 보고하는 ROAS가 아니라 Cafe24 실매출 기준으로 평가하므로, "광고가 진짜 매출을 만들었나"를 현실적으로 판단할 수 있어요.
